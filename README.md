# 🐳 Docker • Nginx • Apache — Simple Study Guide

> A beginner-friendly, no-jargon study guide for Docker, Nginx, and Apache.  
> Built for IT students, self-learners, and anyone tired of LinkedIn tech-splaining.

---

## 📖 Overview

This is a **single HTML5.3+ study guide** that explains:

- 🐳 **Docker** — Containers, images, Dockerfiles, and common commands
- 🦈 **Nginx** — Fast web server, reverse proxy, configuration examples
- 🦁 **Apache** — Reliable web server, .htaccess, PHP hosting

No jargon. Just clear examples you can actually understand.
